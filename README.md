# SECV2223 Online Voting System (Prototype)

A simple **web-based online voting system** prototype built with **PHP + MySQL** (backend) and **HTML/CSS/JavaScript** (frontend).  
Users can **create a voting question/poll**, then **share a Vote ID** for other users to participate and submit votes.  

> Educational prototype / learning project.

---

## What the system does

- Create a voting/poll question
- Generate / share a **Vote ID**
- Other users can enter the Vote ID and submit a vote
- View responses / results (depends on the page flow in the project)

---

## Tech Stack

- Frontend: HTML, CSS, JavaScript  
- Backend: PHP  
- Database: MySQL  

---

## Prerequisites

You need a local server environment that supports PHP + MySQL, such as:

- **XAMPP** (Windows/macOS/Linux)  
- **WAMP** (Windows)  
- **MAMP** (macOS)  
- Any Apache/Nginx + PHP + MySQL setup

Also recommended:
- PHP 7.x/8.x
- MySQL / MariaDB
- phpMyAdmin (usually included with XAMPP/WAMP)
